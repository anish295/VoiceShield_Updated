Project Description

Voice Shield is an AI-powered aggression detection system that combines facial expression analysis and voice emotion recognition to detect aggression levels in real-time.

Key Features

  • Real-time Facial Expression Analysis using face-api.js
  • Voice Emotion Detection and analysis
  • Combined AI Analysis integrating visual and audio cues
  • Real-time WebSocket Communication with Socket.IO
  • Modern, responsive UI with real-time visualizations
  • Comprehensive scoring system for facial, voice, and combined analysis

Technology Stack

Frontend: HTML5, CSS3, JavaScript (ES6+), Tailwind CSS, face-api.js, Web Audio API

Backend: Python 3.8+, Flask, Flask-SocketIO, NumPy, Audio processing libraries
